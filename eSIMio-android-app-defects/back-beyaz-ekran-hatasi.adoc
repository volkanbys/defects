*Test cihazı sürümü:* Android 13

*Test cihazı ekran çözünürlüğü:* 1080 x 2400

*Test cihazı arayüz sürümü:* One UI 5.1

*Uygulama Sürümü:* 1.0.2

*Sorunun Tekrarlanma Tarih-Saati:* 20.12.2023 - 16:52

**Aşama:**

- Anasayfa

**Senaryo:**
